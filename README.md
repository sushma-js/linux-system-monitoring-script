# linux-system-monitoring-script
A Linux shell script that monitors system resources such as CPU usage, memory usage, and disk usage to help administrators track system performance.

# Linux System Monitoring Script

This project contains a simple Bash script to monitor system resources.

## Features
- CPU usage monitoring
- Memory usage monitoring
- Disk usage monitoring

## Technologies Used
- Linux
- Bash (Shell Scripting)

## How to Run

chmod +x monitor.sh
./monitor.sh
